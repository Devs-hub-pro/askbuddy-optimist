# 前端逐页人工验收清单（最终轮）

更新时间：2026-03-07

## 全局规范（所有页面）
- [ ] 顶部安全区：顶部栏不与 iPhone 时钟/电量重叠（含刘海屏）。
- [ ] 固定头部：一级页与核心二级页滚动时顶部保持固定，层级不抖动。
- [ ] 返回路径：二级页返回优先回上一页；无历史时回退到频道/首页兜底。
- [ ] 底部导航：仅一级页显示，二级/三级页面不出现底部导航冲突。
- [ ] 状态页：loading / empty / error 样式、按钮文案、间距统一。
- [ ] 触控反馈：卡片、按钮、胶囊点击区 >= 40px，反馈一致。

## 首页与四大频道
- [ ] 首页：顶部导航 + 搜索条固定；问问热榜以下可滚动。
- [ ] 首页：问题卡片与专家卡片视觉节奏一致（圆角、阴影、字号、按钮）。
- [ ] 教育学习：主题色（蓝）在头部/搜索/高亮元素一致。
- [ ] 职业发展：主题色（绿）在头部/搜索/高亮元素一致。
- [ ] 生活服务：主题色（橙）在头部/搜索/高亮元素一致。
- [ ] 兴趣技能：主题色（粉）在头部/搜索/高亮元素一致。
- [ ] 频道页：横向子类胶囊尺寸/圆角/间距一致，右侧指示器行为正常。
- [ ] 频道页：浮动“+”按钮可拖动、可点击、四页尺寸与阴影一致。

## 搜索链路
- [ ] 首页搜索与四频道搜索均可进入搜索结果页。
- [ ] 搜索框聚焦边框颜色跟随频道主题，不出现默认青绿色串色。
- [ ] 搜索“留学”等关键词可命中问题/专家/话题（含 demo 回退）。
- [ ] 搜索结果页：历史搜索、热门搜索、空结果、结果分栏显示正常。

## 发现 / 消息 / 我的
- [ ] 发现页：顶部色调与站点主色统一；右上角通知进入互动消息页。
- [ ] 消息页：首次进入速度正常；私信与通知切换、已读操作可用。
- [ ] 我的页：入口不重复（订单/设置等）；信息层级清晰。
- [ ] 我的页子链路：编辑资料、收益、社群、认证、帮助等页面头部统一。

## 详情链路
- [ ] 问题详情：标题-正文-作者-操作栏层级清晰，滚动与底栏不冲突。
- [ ] 热榜详情：专题阅读、评论讨论、返回路径一致。
- [ ] 专家链路：专家主页与咨询页职责清晰，进入路径统一。

## 建议复测路径（手工）
1. 首页 -> 频道（教育/职业/生活/兴趣）-> 搜索 -> 返回。
2. 首页 -> 问问热榜 -> 专题详情 -> 评论 -> 返回。
3. 首页/频道 -> 专家卡 -> 专家主页 -> 咨询页 -> 返回。
4. 消息页（私信/通知）-> 已读操作 -> 详情跳转 -> 返回。
5. 我的 -> 设置中心 -> 任意二级页 -> 三级页 -> 返回。
